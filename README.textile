h2. Demo: S3 & canvas & no Security Error

This is a demo app for "blog post":http://dira.ro

h2. You can use the code

See the source directories for the "application":https://github.com/dira/cross-domain-image-proxy/tree/master/application/src and for the "image hosting domain":https://github.com/dira/cross-domain-image-proxy/tree/master/s3/src

h2. Install it on your machine

* clone the repository
* have a web server deliver @application/public@ under the URL @http://application.dev@ (I use "pow":http://pow.cx/)
* have a web server deliver @s3/public@ under the URL @http://s3.dev@ (I use "pow":http://pow.cx/)
* open "http://application.dev":http://application.dev
